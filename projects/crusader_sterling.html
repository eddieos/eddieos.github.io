CrusaderSterling Pension Mobile Application

Overview

The Problem

Crusadersterling was in need of an Hybrid Mobile Application to serve their iOS and Android customers a good pension experience. There was an existing software working on just the Android platform, so there was a need to build a new app that supports both iOS and Android.

What I did

Worked with the business and development team in visualizing how the application should best work for customers. Designed wireframes based on the user stories provided by the business team. After approved wireframes, I joined the dev team in building the interface of the mobile app using Ionic Framework.
